Sheriyans Coding School Landing Page Clone

A fully responsive landing page clone of the Sheriyans Coding School Cohort webpage built using HTML5 and CSS3.

This project was created to practice and strengthen my front-end development skills, focusing on responsive layouts, clean UI design, and pixel-perfect implementation.
